## Main Changes

1. Chat page accessibility semantics
- Replaced ambiguous control narration around chat entry with explicit function labels.
- Covered top bar and bottom bar controls, including model store, local import, model selector, web search toggle, thinking toggle, send, and stop.

2. Component-level fallback semantics
- Implemented icon-to-purpose fallback descriptions in `ActionButtons.kt` to reduce generic narration on icon buttons.

3. First-launch Chinese adaptation improvements
- Updated Guide, Terms header/actions, Setup option cards, Performance mode picker, and Restore-from-backup dialog to use Chinese-aware text path.

4. Model workflow Chinese adaptation improvements
- Updated Model Picker, Model Store tabs/search/filter labels, and basic empty/error states in browse/installed views.

5. Translation dictionary expansion
- Expanded `TnI18n.kt` for user-facing strings used in this release path.
